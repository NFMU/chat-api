# Identity

Identity service scaffold rebuilt with NestJS and TypeORM.

## Environment

Set these environment variables before running locally:

- `PORT`
- `DB_HOST`
- `DB_PORT`
- `DB_USERNAME`
- `DB_PASSWORD`
- `DB_NAME`
- `AUTO_RUN_MIGRATIONS`
- `AUTO_RUN_SEEDS`

## Install

```bash
npm install
```

## Run

```bash
npm run start:dev
```

## Build

```bash
npm run build
```
